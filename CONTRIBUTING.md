# Hướng Dẫn Đóng Góp (Contributing Guide)

Chào mừng bạn đến với dự án **CareerPath AI**! Chúng tôi rất vui khi bạn muốn cùng phát triển dự án này.

## 🛠️ Thiết Lập Môi Trường Phát Triển

1.  **Clone dự án**:
    ```bash
    git clone https://github.com/<username>/CareerPath_AI_Project.git
    cd CareerPath_AI_Project
    ```
2.  **Cài đặt Backend**: Theo hướng dẫn trong [README.md](./README.md#backend-setup).
3.  **Cài đặt Frontend**: Theo hướng dẫn trong [README.md](./README.md#frontend-setup).
4.  **Cấu hình biến môi trường**: Sao chép các file mẫu (nếu có) hoặc tạo file `.env` theo hướng dẫn trong README.

## 🌿 Quy Trình Nhánh (Branching)

Chúng tôi sử dụng mô hình Git Flow đơn giản:
- **`main`**: Nhánh chính chứa mã nguồn ổn định nhất.
- **`develop`**: Nhánh phát triển tích hợp (nếu có).
- **`feature/name`**: Các nhánh chứa tính năng mới.
- **`fix/issue`**: Các nhánh sửa lỗi.

## 📝 Commit Messages

Hãy viết commit message theo phong cách [Conventional Commits](https://www.conventionalcommits.org/):
- `feat: thêm tính năng X`
- `fix: sửa lỗi Y`
- `docs: cập nhật tài liệu`
- `style: thay đổi giao diện/format`

## 🚀 Gửi Đóng Góp

1.  Đảm bảo mã của bạn đã được test local.
2.  Push nhánh của bạn lên GitHub.
3.  Mở một **Pull Request (PR)** trên GitHub.
4.  Mô tả rõ những thay đổi bạn đã thực hiện trong PR.
